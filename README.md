# The Unwind Project

A student wellness hub, built by students for students. This website provides a collection of simple tools, articles, and resources to help college students manage stress, find focus, and access campus support.

**Live Demo:** [https://the-unwind-project1.netlify.app/](https://the-unwind-project1.netlify.app/)

## âœ¨ Features

* **Guided Breathing Exercise:** A simple, full-screen visualizer ("Just Breathe") to help users calm down and practice mindfulness.
* **Quick Reads:** A collection of short, helpful articles on student-centric topics like "5-Minute Mindfulness," "Beating Procrastination," and "Dorm Room Workouts."
* **Campus Resource Guide:** A filterable list of essential campus services (e.g., Counseling, Health Services, Academic Support) with locations and contact information.
* **Clean, Focused UI:** A simple and calming interface designed to be a relaxing and supportive space for students.
